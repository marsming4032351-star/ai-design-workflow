## 基础概念

### 什么是智能体 (Agent)?

智能体是一个具有特定职能和专业知识的 AI 助手。本系统包含 6 个智能体,每个负责设计流程的一个环节。

### 什么是提示词框架?

提示词框架是结构化的指令模板,确保 AI 输出的专业性和一致性。本系统使用 6 种框架:

- **BROKE**: 策略分析
- **COSTAR**: 彩平绘制
- **RTGO**: 3D 建模
- **CARE**: 风格控制
- **RTF**: 摄影渲染
- **ROSES**: 方案排版

### 什么是 TRACE 学习框架?

TRACE 是案例学习系统,可以从优秀设计案例中提取知识并应用到新项目:

- **T**ask: 提取设计语言
- **R**equest: 量化参数
- **A**ction: 生成知识卡片
- **C**ontext: 关联智能体
- **E**xample: 参考样张

## 使用逻辑

### 三种使用模式

#### 模式 1: 单点使用

只需要某个特定功能:

```
# 只要彩平图
"请为这个 CAD 图纸生成彩色平面图"

# 只要项目分析
"请帮我分析这个咖啡馆项目"

# 只要 3D 效果图
"请生成客厅视角的 3D 透视图"
```

#### 模式 2: 流程使用

完整的 6 步设计流程:

```
步骤 1: 方案资讯官 - 项目分析
步骤 2: 彩平绘制师 - 生成彩平图
步骤 3: 三维建模师 - 生成 3D 图
步骤 4: 视觉艺术家 - 优化风格
步骤 5: 杂志摄影师 - 摄影渲染
步骤 6: 软装排版师 - 方案交付
```

#### 模式 3: 学习使用

学习案例并应用:

```
步骤 1: 提供案例 (链接/图片/描述)
步骤 2: 系统自动学习 (TRACE 框架)
步骤 3: 生成知识卡片
步骤 4: 应用到新项目
```

## 6 个智能体详解

### 1. 方案资讯官 (design-analyst)

**何时使用**: 项目开始阶段,需要分析和策划时

**输入内容**:
- 项目基本信息(类型、面积、位置)
- 目标人群
- 预算范围
- 竞争对手信息

**使用示例**:
```
请帮我分析一个项目:
- 类型: 精品咖啡馆
- 面积: 150 平米
- 位置: 商务区写字楼一层
- 目标: 25-40 岁白领
- 预算: 中高端
```

**输出内容**:
- 项目背景分析
- 用户画像
- 设计策略
- KPI 指标
- 演变路径

---

### 2. 彩平绘制师 (design-draftsman)

**何时使用**: 有 CAD 图纸,需要生成彩色平面图时

**输入内容**:
- CAD 图纸 (上传图片)
- 风格要求
- 材质偏好

**使用示例**:
```
请为这个 CAD 图纸生成彩色平面图:
- 风格: 现代简约
- 地面: 木地板
- 墙面: 白色涂料
- 家具: 浅木色
```

**输出内容**:
- 高质量彩色平面图
- 材质和标注
- 动线引导

**技巧**:
- 可以指定具体材质和颜色
- 可以要求不同风格的版本
- 可以调整阴影和光线方向

---

### 3. 三维建模师 (design-architect)

**何时使用**: 需要将平面图转换为 3D 效果图时

**输入内容**:
- 彩色平面图
- 视角要求
- 层高信息

**使用示例**:
```
请生成 3D 透视图:
- 视角: 客厅入口视角
- 相机高度: 1.6m (人眼)
- 层高: 3.0m
- 焦段: 24mm 广角
```

**输出内容**:
- 3D 透视效果图
- 准确的空间体量
- 真实的透视关系

**视角选择**:
- 人眼视角 (1.5-1.7m): 最真实
- 鸟瞰视角 (3-5m): 展示布局
- 低角度 (0.8-1.2m): 强调高度
- 角落视角: 展示多个墙面

---

### 4. 视觉艺术家 (design-stylist)

**何时使用**: 需要调整材质、灯光或风格时

**输入内容**:
- 3D 效果图
- 风格要求
- 氛围描述

**使用示例**:
```
请优化这个效果图:
- 风格: 北欧风格
- 氛围: 温暖、明亮
- 时间: 早晨阳光
- 色温: 3500K
- 材质: 浅木色为主
```

**输出内容**:
- 优化后的效果图
- 统一的风格
- 和谐的色彩
- 真实的材质

**可调整内容**:
- 地面/墙面/天花材质
- 自然光和人工光
- 色彩搭配
- 装饰元素

---

### 5. 杂志摄影师 (design-photographer)

**何时使用**: 需要杂志级质量的最终效果图时

**输入内容**:
- 风格化效果图
- 摄影参数要求
- 构图偏好

**使用示例**:
```
请生成杂志级照片:
- 焦段: 50mm 标准镜头
- 光圈: f/2.8 (适中景深)
- 构图: 三分法则
- 后期: 专业色彩校正
```

**输出内容**:
- 杂志级摄影渲染
- 专业的构图
- 精确的景深控制
- 完美的光影

**焦段选择**:
- 16-24mm: 展示空间感
- 35mm: 自然全景
- 50mm: 真实视角
- 85mm: 细节特写

**光圈选择**:
- f/1.4-f/2.0: 极浅景深,艺术化
- f/2.8-f/4: 适中景深,专业
- f/5.6-f/8: 较深景深,清晰
- f/11-f/16: 极深景深,全景

---

### 6. 软装排版师 (design-curator)

**何时使用**: 需要生成方案册和物料清单时

**输入内容**:
- 摄影级效果图
- 项目信息
- 交付格式要求

**使用示例**:
```
请生成完整的方案交付文档:
- 方案册 (PDF)
- 物料清单 (Excel)
- 效果图集 (JPG)
- 预算汇总
```

**输出内容**:
- 专业方案册
- 详细物料清单
- 预算汇总表
- 多格式文件

**包含内容**:
- 封面和目录
- 项目概况
- 设计说明
- 效果图展示
- 平面布置图
- 物料清单
- 预算表

## 完整项目流程

### 案例: 150 平米咖啡馆设计

#### 步骤 1: 项目分析 (方案资讯官)

```
请帮我分析一个咖啡馆项目:
- 面积: 150 平米
- 位置: 商务区写字楼一层
- 目标: 25-40 岁白领,追求品质
- 预算: 8000-12000 元/平米
- 竞争: 周边有星巴克、瑞幸
```

**输出**: 策略报告,建议定位为"第三空间",强调舒适和私密性

#### 步骤 2: 生成彩平图 (彩平绘制师)

```
请为这个 CAD 图纸生成彩色平面图:
- 风格: 现代简约
- 地面: 浅色木地板
- 墙面: 白色+深灰点缀
- 家具: 木色+灰色布艺
```

**输出**: 专业彩色平面图,清晰的功能分区

#### 步骤 3: 生成 3D 效果图 (三维建模师)

```
请生成 3 个角度的 3D 效果图:
1. 入口视角 (24mm 广角)
2. 吧台区视角 (35mm)
3. 座位区视角 (50mm)
```

**输出**: 3 个角度的 3D 透视图

#### 步骤 4: 优化风格 (视觉艺术家)

```
请优化效果图:
- 时间: 下午 3 点柔和阳光
- 氛围: 温暖、舒适、放松
- 色温: 3500K
- 增加绿植和装饰画
```

**输出**: 温暖舒适的优化效果图

#### 步骤 5: 摄影渲染 (杂志摄影师)

```
请生成杂志级照片:
- 焦段: 35mm
- 光圈: f/4
- 构图: 三分法则
- 后期: 专业校色
```

**输出**: 杂志级最终效果图

#### 步骤 6: 方案交付 (软装排版师)

```
请生成完整方案:
- 方案册 (包含所有效果图和说明)
- 物料清单 (家具、灯具、装饰品)
- 预算表 (详细报价)
```

**输出**: 完整的项目交付文档

## 高级技巧

### 1. 精确参数控制

```
请生成效果图,使用以下精确参数:
- 地面: 深灰剁斧石,#4A5568,哑光
- 天花: 橡木格栅,50mm 间距,#D4A574
- 灯光: 3000K 主光 + 2700K 重点光
- 相机: 24mm, f/5.6, 1.6m 高度
```

### 2. 多版本对比

```
请生成 3 个版本的效果图:
版本 A: 现代简约风格
版本 B: 北欧风格
版本 C: 工业风格
保持布局不变,只调整材质和色彩
```

### 3. 局部优化

```
请只优化这个效果图的灯光:
- 保持材质和布局不变
- 调整为黄昏氛围
- 色温 2700K
- 增加暖色调
```

### 4. 风格迁移

```
请将这个现代风格的空间:
- 转换为岁集风格
- 保持家具布局
- 更换地面为深灰石材
- 更换天花为木格栅
- 调整灯光为 3000K
```

### 5. 细节特写

```
请为这个空间生成 3 张细节特写:
1. 吧台区域 (85mm, f/2.8)
2. 座位区装饰画 (85mm, f/2.0)
3. 绿植角落 (50mm, f/4)
```

## 常见问题

### Q: 如何获得最佳效果?

A: 
1. 提供清晰的需求和参数
2. 使用具体的数值而非模糊描述
3. 参考已学习的案例风格
4. 逐步优化而非一次到位

### Q: 可以同时使用多个智能体吗?

A: 建议按顺序使用,每个智能体的输出作为下一个的输入,确保信息传递完整。

### Q: 如何调整不满意的效果?

A:
1. 明确指出不满意的部分
2. 提供具体的调整方向
3. 可以要求局部优化
4. 可以参考其他案例风格

### Q: 学习案例需要多长时间?

A: 通常几分钟内完成,系统会自动提取参数并生成知识卡片。

### Q: 可以学习自己的项目作为案例吗?

A: 可以!任何优秀的设计都可以作为学习案例。
