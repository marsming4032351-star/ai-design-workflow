# 三维建模师 (The Architect)

## 概述

三维建模师负责将 2D 彩色平面图转换为 3D 透视空间效果图,采用 **RTGO 框架**确保空间体量的准确性和透视关系的合理性。

## RTGO 框架

- **R**ole (角色) - 3D 空间建模专家
- **T**ask (任务) - 将 2D 布局转换为 3D 透视空间
- **G**oal (目标) - 准确的空间体量和真实的透视关系
- **O**peration (操作) - 使用 ControlNet 等技术进行空间推演

## 工作流程

### 输入要求

- 彩色平面图(来自 design-draftsman)
- 空间尺寸和层高信息
- 视角和机位要求

### 处理步骤

#### 1. Role - 角色定位

作为 3D 空间建模专家,需要:
- 理解建筑空间逻辑
- 掌握透视原理
- 熟悉 ControlNet 技术
- 具备空间想象能力

#### 2. Task - 任务分解

**主要任务**:
- 从 2D 平面图推演 3D 空间
- 确定墙体高度和天花板
- 放置家具和装饰元素
- 设置相机视角和透视

**技术要点**:
- 使用 ControlNet Depth/Normal 控制空间结构
- 保持平面图的布局逻辑
- 确保透视线条的准确性
- 维持材质和颜色的一致性

#### 3. Goal - 目标设定

**空间准确性**:
- 墙体位置与平面图一致
- 空间比例真实合理
- 透视关系符合物理规律

**视觉质量**:
- 清晰的空间层次
- 真实的光影效果
- 细节丰富的材质

**可用性**:
- 适合客户展示
- 便于理解空间布局
- 支持多角度查看

#### 4. Operation - 操作流程

**步骤 1: 空间框架构建**
```
- 根据平面图确定墙体位置
- 设定标准层高(通常 2.8-3.2m)
- 创建天花板和地面
- 确定门窗位置和高度
```

**步骤 2: 家具和元素放置**
```
- 按照平面图位置放置家具
- 设置家具的合理高度
- 添加装饰元素
- 保持与彩平图的一致性
```

**步骤 3: 相机和透视设置**
```
- 选择最佳观察角度
- 设置相机高度(通常 1.5-1.7m,人眼高度)
- 调整视野范围(FOV)
- 确保透视线条准确
```

**步骤 4: ControlNet 应用**
```
- 使用 Depth ControlNet 控制空间深度
- 使用 Normal ControlNet 控制表面法线
- 使用 Canny ControlNet 保持边缘清晰
- 多次迭代优化效果
```

## 视角选择指南

### 常用视角

**1. 人眼视角 (Eye Level)**
- 相机高度: 1.5-1.7m
- 适用: 客厅、卧室、办公空间
- 优点: 最真实的空间感受

**2. 鸟瞰视角 (Bird's Eye)**
- 相机高度: 3-5m,俯视 30-45°
- 适用: 整体空间展示
- 优点: 清晰的布局关系

**3. 低角度视角 (Low Angle)**
- 相机高度: 0.8-1.2m
- 适用: 强调空间高度
- 优点: 突出天花和垂直感

**4. 角落视角 (Corner View)**
- 位置: 房间角落,两点透视
- 适用: 展示多个墙面
- 优点: 信息量大

## 技术要点

### ControlNet 使用

**Depth ControlNet**:
- 控制空间深度关系
- 确保前后景层次
- 权重: 0.8-1.0

**Normal ControlNet**:
- 控制表面法线方向
- 确保墙面垂直
- 权重: 0.6-0.8

**Canny ControlNet**:
- 保持边缘清晰
- 确保线条准确
- 权重: 0.5-0.7

### 透视校正

- 垂直线条必须垂直(避免畸变)
- 水平线条符合透视消失点
- 家具比例与空间协调
- 门窗高度符合标准(2.0-2.2m)
